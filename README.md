Nothing (Concept) Landing Page.

A responsive landing page inspired by the **Nothing** brand aesthetic.

**[LIVE DEMO](https://dianakerekesha.github.io/Simple-Step-Landing/)**

---

##esign Philosophy
The **Nothing** design is based on transparency and minimalism. Key features implemented:
- Signature **Dot Matrix** typography.
- Monochrome color palette.
- High attention to white space and clean layouts.

---

## Tech Stack

I used a modern development toolkit to bring this project to life:

* **HTML5 / SCSS** — Semantic structure and modular style architecture.
* **Gulp** — Automation for Sass compilation, autoprefixing, and local development server.
* **Stylelint** — Strict linting to ensure CSS code quality and standards.
* **BEM (Block Element Modifier)** — Naming convention for scalable and readable code.
* **GitHub Pages** — Automated deployment for the live demo.

Clone this repository:

Bash

Navigate to the project directory:

Bash
cd Simple-Step-Landing
Install the required dependencies (ensure you use the recommended Node.js version):

Bash
npm install
Start the development mode with live reloading:

Bash
npm start
